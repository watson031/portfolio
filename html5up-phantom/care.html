Care Project

A web and mobile application that helps follow up with a patient after they have had an operation. Indeed, after a post-operative operation, a patient must register with a representative to receive an access code that will allow him to connect to the mobile application. The latter allows the user to send a photo to the doctor or the practitioner, send a message when there is a concern and answer daily questions for a proper follow-up.

Technologies used : Java Android, React Js, NodeJs, Express, Android Studio, WebStorm, Visual Studio Code, PostgreSQL, Heroku, AWS (S3), Agile Methodologie

Space Invaders Project

Space Invaders game that allows the appearance of Martians from which we must defend ourselves by throwing lasers at them.There is a score that increases depending on the number of Martians killed. We win the game if we manage to reach a certain number of points or if we succeed in eliminating all the Martians on the console.

Technologies used : C++, UIKIT, Visual Studio, Bitbucket
